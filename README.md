## Overview

Cows vs Humans is a retro-style 2D shooter inspired by Space Invaders.  
You control a UFO cow defending the skies from waves of human invaders.  
Blast enemies, earn points, and survive as long as you can.

---

## Features

- Smooth player movement and responsive controls  
- Ten human enemies descending in waves  
- Collision detection using Euclidean distance  
- Live scoring system  
- Game over screen when enemies reach the ground  
- Customizable assets (replace with your own sprites)  
- Built entirely in under 250 lines of Python
